Master’s thesis

Pedestrian Intention Prediction for an Autonomous Delivery Robot

Research context

Human-Aware Autonomous Navigation

My thesis focused on improving the safety of an autonomous grocery delivery robot operating in environments shared with pedestrians.

The objective was to help the robot understand current pedestrian behavior and anticipate possible short-term changes so that it could support safer and more risk-aware path planning.

Technical approach

Perception and Prediction Pipeline

  • Pedestrian detection and tracking
  • Depth camera, GPS and odometry data
  • Action recognition using an I3D model
  • LSTM-based prediction of action-state changes
  • MLP layer for intention probability estimation
  • Risk-aware path-planning support
